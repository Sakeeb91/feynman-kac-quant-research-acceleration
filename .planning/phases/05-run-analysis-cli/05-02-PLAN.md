`emit_comparison_table(comparison: dict, verbose: bool = False, console: Console | None = None) -> None`:
- If no console, use `Console(stderr=True)`.
- Print summary panel at top (using Rich Panel): "Matched: {N} | Only in A: {N} | Only in B: {N} | A wins: {N} | B wins: {N}".

`@app.command("compare-runs")` with parameters:
- `run_a: str = typer.Argument(..., help="First run (UUID, prefix, latest, latest~N)")`
- `run_b: str = typer.Argument(..., help="Second run (UUID, prefix, latest, latest~N)")`
- `db_path: str = typer.Option("artifacts/experiments.db", "--db-path")`
- `all_status: bool = typer.Option(False, "--all-status", help="Include non-completed scenarios")`
- `output_format: str | None = typer.Option(None, "--format")`
- `verbose: bool = typer.Option(False, "--verbose")`

Implementation:
1. Open MetadataStore(db_path).
2. Resolve both run_a and run_b via `resolve_run_id`. Wrap in try/except ValueError -> `log.error(...)` + `typer.Exit(code=1)`.
3. Call `compute_comparison(store, resolved_a, resolved_b, include_all_status=all_status)`.
4. Determine format via `get_effective_format(output_format)`.
5. Dispatch to comparison table/json/csv emitter.
6. Close store.

`@app.command("show-run")` with parameters:
- `run_id: str = typer.Argument(..., help="Run to inspect (UUID, prefix, latest, latest~N)")`
- `db_path: str = typer.Option("artifacts/experiments.db", "--db-path")`
- `output_format: str | None = typer.Option(None, "--format")`
- `verbose: bool = typer.Option(False, "--verbose")`

Implementation:
1. Open MetadataStore(db_path).
2. Resolve run_id via `resolve_run_id`. Handle ValueError.
3. Fetch batch run via `store.get_batch_run(resolved_id)`. If None, error + exit.
4. Fetch scenario runs via `store.get_scenario_runs(resolved_id)`.
5. Determine format and dispatch to show-run table/json/csv emitter.
6. Close store.
