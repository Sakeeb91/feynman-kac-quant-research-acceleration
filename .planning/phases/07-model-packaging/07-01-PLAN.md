Create `src/fk_quant_research_accel/packaging/assembler.py`:

```python
class ModelPackager:
    def __init__(self, store: MetadataStore, artifacts_dir: str | Path) -> None:
        ...

    def export_package(
        self,
        batch_run_id: str,
        output_dir: str | Path,
        scenario_run_id: str | None = None,
        force: bool = False,
    ) -> Path:
        ...
```

Implementation steps in `export_package`:
1. `store.get_batch_run(batch_run_id)` -- get batch metadata (config_json, seed, git_sha, python_version, os_info, problem_id)
2. Find winning scenario: if `scenario_run_id` given, find that specific one; otherwise `min(completed, key=score)` from `store.get_scenario_runs(batch_run_id)`
3. Parse JSON columns: `json.loads()` on `result_json`, `scenario_json`, `config_json` (following Pitfall 3)
4. Read `manifest.yaml` from `artifacts_dir/{batch_run_id}/manifest.yaml` for reproducibility info (packages dict, etc.) -- per Pitfall 5, do NOT call `capture_environment()`
5. Create package directory: `output_dir/model_pkg_{batch[:8]}_{scenario[:8]}/`
   - If exists and not `force`: raise `FileExistsError`
   - If exists and `force`: `shutil.rmtree` then recreate
6. Copy checkpoint: check `scenario_row["checkpoint_path"]`, verify file exists on disk, copy with `shutil.copy2`, compute SHA-256
7. Write config files: `training_config.yaml`, `scenario_config.yaml` via `yaml.safe_dump(data, sort_keys=True)`
8. Write environment: `reproducibility.yaml` from manifest data, `seed.txt` if seed is not None
9. Run acceptance checks: `check_acceptance(metrics_dict, convergence_health, checkpoint_path)`
10. Write validation: `metrics.yaml`, `acceptance.yaml`
11. Generate `README.txt`: template with batch_run_id, scenario_run_id, problem_id, score, convergence_health, acceptance status, created_at timestamp
12. Collect all relative file paths in the package directory
13. Build `ModelPackageManifest`, write as `MANIFEST.yaml`
14. Return package directory path

Update `src/fk_quant_research_accel/packaging/__init__.py` to also export `ModelPackager`.
