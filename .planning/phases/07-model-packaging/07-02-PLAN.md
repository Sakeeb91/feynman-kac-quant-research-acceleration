```python
@app.command("export-model")
def export_model_command(
    run_id: str = typer.Argument(
        ..., help="Batch run to export (UUID, prefix, latest, latest~N)"
    ),
    output_dir: str = typer.Option(
        "model_packages", "--output-dir",
        help="Directory to create the package in",
    ),
    scenario_id: str | None = typer.Option(
        None, "--scenario-id",
        help="Export a specific scenario (default: best-scoring)",
    ),
    db_path: str = typer.Option(
        "artifacts/experiments.db", "--db-path",
    ),
    artifacts_dir: str = typer.Option(
        "artifacts", "--artifacts-dir",
    ),
    force: bool = typer.Option(
        False, "--force",
        help="Overwrite existing package directory",
    ),
    zip_package: bool = typer.Option(
        False, "--zip",
        help="Compress package into a .zip archive",
    ),
) -> None:
```

Implementation:
1. Open `MetadataStore(db_path)` in try/finally pattern (matching show-run, list-runs)
2. `resolved = resolve_run_id(run_id, store)` -- reuse existing resolver
3. Create `ModelPackager(store=store, artifacts_dir=artifacts_dir)`
4. Call `package_path = packager.export_package(resolved, output_dir, scenario_run_id=scenario_id, force=force)`
5. If `zip_package` is True:
   - Use `shutil.make_archive(str(package_path), 'zip', root_dir=str(package_path.parent), base_dir=package_path.name)`
   - Log the zip path
   - Optionally `shutil.rmtree(package_path)` after zipping
6. Display Rich summary to stderr (matching leaderboard pattern):
   - Use `rich.console.Console(stderr=True)` and `rich.table.Table`
   - Show: Package path, Batch run ID, Scenario run ID, Problem ID, Score, Convergence health, Acceptance status (PASSED/FAILED)
   - Use green for PASSED, red for FAILED
7. Handle errors:
   - `ValueError` (no run found, no completed scenarios): `log.error(...)`, `raise typer.Exit(code=1)`
   - `FileExistsError` (package dir exists without --force): `log.error("package_exists", ...)`, `raise typer.Exit(code=1)`
   - `FileNotFoundError` (manifest.yaml missing): `log.error(...)`, `raise typer.Exit(code=1)`
